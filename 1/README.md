## Безопасный файловый менеджер

Для работы нужно сначала зарегистрировать пользователя и войти под ним. Есть все функции для работы с zip, json, xml с предварительной проверкой файлов. Работа происходит в `storage` директории, там где была выполнена программа. Действия сохраняются в sqlite базу данных `filemanager.db`, все устойчиво к sql инъекциям благодаря встроенной библиотеке для Bun(JavaScript). Совершая действия над файлом, он блокируется для оставльных пользователей, кроме текущего.

### Запуск

-   Установить Bun: https://bun.sh/

-   Установить зависимости и запустить:

```bash
bun install
```

-   Запуск

```bash
bun start
```
